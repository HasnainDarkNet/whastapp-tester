# WhatsApp-Test (Educational Demo)

## ⚠️ Disclaimer

This project is created **only for educational, UI/UX learning, Flask practice, and cybersecurity awareness purposes**.

* Do NOT use this project to collect real phone numbers, OTPs, passwords, or personal information.
* Do NOT deploy this project against real users.
* The author is not responsible for any misuse of this code.
* All demonstrations should be performed in a controlled lab environment using test data only.

---

## 📖 About

WhatsApp-Test is a Flask-based educational project that demonstrates:

* Multi-step authentication UI design
* Flask routing and form handling
* OTP interface implementation
* Frontend form validation
* Responsive web design
* Cybersecurity awareness demonstrations

This project can be used by students and beginner developers to understand how verification workflows are designed in modern web applications.

## 📦 Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/whatsapp-test.git

cd whatsapp-test
```

### Create Virtual Environment

Windows:

```bash
python -m venv venv

venv\Scripts\activate
```

Linux/macOS:

```bash
python3 -m venv venv

source venv/bin/activate
```

### Install Requirements

```bash
pip install -r requirements.txt
```

---

## ▶️ Run Application

```bash
python app.py
```
